# MirrorPPR Project Page

Static GitHub Pages project page for **MirrorPPR: Exemplar-Based Portrait Photo Retouching**.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8124 --bind 127.0.0.1
```

then visit `http://127.0.0.1:8124/`.

## GitHub Pages

1. Create a public GitHub repository.
2. Push this folder as the repository root.
3. In GitHub, open `Settings -> Pages`.
4. Select `Deploy from a branch`, branch `main`, folder `/(root)`.
5. Replace the placeholder `arXiv`, `Code`, `Model`, and `Dataset` links in `index.html` after those resources are released.
